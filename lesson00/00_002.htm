Auto Play Setting

This eLearning is programmed to automatically play content when you enter a page. If you would prefer to manually play each page, select the “No” option below:

In addition to the buttons on the player, keyboard shortcuts can be used to control these settings:

Operating System Browser Pause or Play Content
Windows Internet Explorer, Chrome, or Safari [Alt] + j
Firefox [Alt] + [Shift] + j
Mac Chrome, Firefox, or Safari [Control] + [Alt] + j

Audio Setting

Essential instructional content is provided by narration, so the player sound is set to "Unmute." If you set the player's audio to "Mute," closed captions will display a transcription of the narration. If you prefer to always display captions, select the "Yes" option below:

In addition to the buttons on the player, keyboard shortcuts can be used to control these settings:

Operating System Browser Mute or Unmute Audio
Windows Internet Explorer, Chrome, or Safari [Alt] + m
Firefox [Alt] + [Shift] + m
Mac Chrome, Firefox, or Safari [Control] + [Alt] + m

At any time during the training, you can change these options by selecting “Settings” at the top of the user interface.
