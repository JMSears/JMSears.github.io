System Requirements

The user interface is designed to be viewed at a screen resolution of 1024 x 768 or higher for optimum usability.

For lower resolution screens, scrolling may be required to view screen contents.

Top of Screen Interface

The top of the screen interface has controls and indicators for the learner interface. Learn about each of these using the illustration shown as a reference-locator for the following functional descriptions.

  1. Lesson Title and Topic Title: The Lesson you are reviewing and current Topic are identified. The example shown here indicates “Article 50: Employee Assistance” as the Lesson and “What is Article 50?” as the current Topic.
  2. Settings Button (Alt+A): Allows you to customize the Lesson’s functionality. Specifically, configuration of Assistive Technology, Auto-Play, and Closed Captions.
  3. Exit Button(Alt+0): Closes the Lesson window. If you exit the Lesson before it is completed, you’ll be returned to your previous location when logging back into the Lesson.
  4. Menu Button(Alt+1): The Menu button provides direct links to each Topic in the Lesson. A checkmark will appear next to the Topic once the requirements for that Topic have been met. The Lesson Exit link will remain unavailable until all Topics have been completed.
  5. Pagination: The pagination indicator shows the page you are currently on, the total number of pages within the Lesson, and the percentage of the Lesson you have completed.

Bottom of Screen Interface 1

The bottom of the screen interface has controls, indicators, and resources for the learner interface. Learn about each of these using the illustration shown as a reference-locator for the following functional descriptions.

  1. Page Progress Bar: A visual indicator of your progress through the current page.
  2. Media Player Control Buttons: Standard media player controls to play, pause, rewind, or fast forward the page content. The speaker icon allows you to control the audio's volume and muting. When muted, closed captioning automatically displays. You may toggle closed captioning on and off by selecting the gear icon. This control is disabled if your sound is muted.
  3. Page Timer: Shows elapsed and total time of the current page.
  4. Lesson Progress Bar: A visual indicator of your progress through the current Lesson.

Bottom of Screen Interface 2

The bottom of the screen interface has controls, indicators, and resources for the learner interface. Learn about each of these using the illustration shown as a reference-locator for the following functional descriptions.

  1. Help Button (Alt+2): The Help button will open this Help Topic at any time.
  2. Resources Button (Alt+3): Selecting this button will open an additional screen containing additional reference material, such as the Glossary.
  3. Master Agreement Button (Alt+M): Opens the Master Agreement Reference Tool providing an interactive version of the 2017 Master Agreement between the VA and NAGE, available for reference at any time.
  4. Page Prompt: Provides instructions on how to interact with the current page.
  5. Navigation Buttons: Select Next button (Alt+5) to navigate to next page and the Back button (Alt+4) to return to the previously viewed page.
