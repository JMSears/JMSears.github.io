Below is a list of included features to support the accessible use of this course:

  • Course Transcript: This course includes a transcript that you can use to review the text of the course at your own pace. Though you are welcome to retake this course as you need, feel free to print the transcript for your personal reference.
  • Closed Captions: This course contains a synchronized text version of the audio narration. By default, this feature is off when you start the course.
  • Accessibility Notes: For slides that may require additional information to be used accessibly, there will be a small set of instructions readable only to screen readers. These instructions are intended to provide additional context for animation or specific activities that the user may be called upon to do.
